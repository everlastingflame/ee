One of several [[Discrete Distributions]], where this uses a discrete random variable $X$ can take on any any finite set of $n$ values, where each variable has the same probability $\frac{1}{n}$

$$
\begin{align}
f(x) = \frac{1}{n}, \text{for } \, x=1,2,\dots ,n \\
\mu = \frac{n+1}{2}, \, \sigma^2 = \frac{n^2-1}{12} \\
\sigma^2 = \text{variance}, \, \mu= \text{mean}
\end{align}
$$

Example: Rolling a cubic die 
$$
\begin{align}
n = 6 \\
\mu = 3.5 \\
\sigma^2 = \frac{35}{12}
\end{align}
$$
